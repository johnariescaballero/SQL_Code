Nashville Housing: End-to-End SQL Project

Project Overview
A deep-dive into the Nashville housing market, focusing on transforming raw data into a clean, analysis-ready schema and performing exploratory analysis to identify market trends.

Project Phases
[Completed] Phase 1: Data Cleaning & Transformation
      Resolved BOM encoding on primary keys.
      Populated missing addresses using self-joins on Parcel IDs.
      Split complex address strings into Address, City, and State.

[In Progress] Phase 2: Exploratory Data Analysis (EDA)
      Identifying price trends by neighborhood.
      Analyzing sales volume fluctuations over time.
