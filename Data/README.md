## Dataset Overview: Nashville Housing
This folder contains the raw data utilized for the cleaning and analysis project.

| Column | Description |
| :--- | :--- |
| **UniqueID** | Unique identifier for each sale record |
| **ParcelID** | Identifier for the specific plot of land |
| **SalePrice** | The price at which the property was sold |
| **Acreage** | The total land area of the property |
